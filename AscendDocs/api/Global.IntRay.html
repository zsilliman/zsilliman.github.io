    Class IntRay

    IntRay is a 2D ray defined by an integer origin, but a float-based direction

    • Note that the direction is not normalized.
    • The start is not necessarily the origin, but rather origin + direction* t_range.x
    • The origin + direction* t_range.y is the end of the ray
