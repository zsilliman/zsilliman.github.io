    ApproximateOverlapsBounds(BoundsInt, Vector2)

    This is an imperfect collision used to check whether a boundsInt overlaps the circle. It checks whether the circle contains one of the corners of the bounds, or the bounds contains the center of the circle. No edge detection happens for performance reasons.
