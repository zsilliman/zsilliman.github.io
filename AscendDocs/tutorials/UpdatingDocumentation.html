    Updating Documentation

    Setup

    • Ensure you have two repositories on your local machine.
      • https://github.com/zsilliman/zsilliman.github.io
      • https://github.com/zsilliman/Ascend
    • From the Ascend repo, navigate to the Documentation folder.
    • Execute the following command: docfx "<path to DocumentationFolder>/docfx.json" -o "<path to root directory of zsilliman.github.io repo>"
    • That will generate all the static HTML documentation for the C# classes and manual markdown tutorials.

    Editing Manual Tutorials

    • Navigate to Documentation/tutorials
    • Edit the .md files as needed
    • Edit the toc.yml file to list any newly added or removed .md files
    • Then run the following docfx command: docfx "<path to DocumentationFolder>/docfx.json" -o "<path to root directory of zsilliman.github.io repo>"
